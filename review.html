번역 검수 도구

원본과 번역을 붙여넣고, 필요한 검사만 켜서 빠르게 오류를 찾으십시오. (멀티셀 지원, TSV 내보내기 지원)

실행
입력 방식
검사
내보내기
통계 행 0 PH 0 NUM 0 LANG 0
원본 · 번역 입력
멀티셀: 셀 사이에 빈 줄 2개 이상을 두십시오
왼쪽: 원본 셀 0
오른쪽: 번역 셀 0
  1. 입력 방식을 선택하십시오. 처음이라면 멀티셀을 권장합니다.
  2. 왼쪽에 원본, 오른쪽에 번역을 붙여넣으십시오.
  3. 필요한 검사 항목만 켜십시오. (플레이스홀더, 숫자, 숨은 언어)
  4. 분석을 누르십시오.
  5. 오른쪽 패널에서 항목의 이동을 눌러, 해당 줄로 즉시 점프하십시오.
멀티셀 참고: 빈 줄이 2줄 이상 연속되는 지점을 셀 경계로 봅니다. 셀 순서대로 매칭됩니다.
언어(문자) 탐지 참고: “언어 판별”이 아니라 “문자 종류(스크립트)” 기반입니다. 한글/태국어 같은 숨은 문자는 매우 정확히 잡힙니다.
